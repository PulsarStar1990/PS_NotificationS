# Напоминалка для Windows 11

Простое приложение-напоминалка на Python с использованием tkinter и SQLite.

## Особенности

- Добавление, просмотр и удаление напоминаний
- Всплывающие окна поверх всех окон при срабатывании напоминания
- Системные уведомления Windows 11 (работают даже при свёрнутом окне)
- Хранение данных в локальной базе SQLite
- Автоматическая проверка просроченных напоминаний
- Статусы: Ожидает, Готово, Просрочено, Отменено

## Требования

- Windows 10/11
- Python 3.7+

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python main.py
```

## Использование

### Добавление напоминания

1. Заполните поля в верхней части окна:
   - **Заголовок** (обязательно)
   - **Описание** (необязательно)
   - **Дата и время** (формат: ГГГГ-ММ-ДД ЧЧ:ММ:СС)
2. Нажмите кнопку **"Добавить"**
3. Поля ввода автоматически очищаются

### Редактирование напоминания

1. Выберите напоминание в таблице
2. Нажмите кнопку **"Обновить"**
3. В открывшемся окне измените данные
4. Нажмите **"Сохранить"**

### Удаление напоминания

1. Выберите напоминание в таблице
2. Нажмите кнопку **"Удалить"**
3. Подтвердите действие в диалоговом окне

### Строка состояния

В нижней части окна отображается цветная строка состояния:

| Событие | Сообщение | Цвет |
|---------|-----------|------|
| Добавление напоминания | "Напоминание добавлено" | 🟢 Зелёный |
| Удаление напоминания | "Напоминание удалено" | 🟡 Жёлтый |
| Обновление напоминания | "Напоминание обновлено" | 🟢 Зелёный |
| Ошибка валидации | "Ошибка: ..." | 🔴 Красный |

### Обновление списка

Нажмите кнопку **"Обновить список"** для перезагрузки данных из БД.

### Статусы напоминаний

| Статус | Описание |
|--------|----------|
| **Ожидает** | Напоминание активно, ожидает срабатывания |
| **Готово** | Задача выполнена пользователем |
| **Просрочено** | Время вышло, напоминание не было выполнено |
| **Отменено** | Пользователь отменил напоминание |

## Работа уведомлений

Приложение проверяет напоминания каждые 30 секунд в фоновом потоке. При наступлении времени:

1. Отображается popup-окно поверх всех окон
2. При закрытии popup — системное уведомление Windows 11
3. Если окно свернуто — работает только системное уведомление

## База данных

При первом запуске автоматически создаётся файл `reminders.db` со следующей структурой:

```sql
CREATE TABLE reminders (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    description TEXT,
    scheduled_at TEXT NOT NULL,
    status TEXT DEFAULT 'Ожидает'
)
```

## Примеры дат

- `2025-12-31 23:59:59` — 31 декабря 2025, 23:59:59
