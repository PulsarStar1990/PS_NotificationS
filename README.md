# Напоминалка для Windows 11

Простое приложение-напоминалка на Python с использованием tkinter и SQLite.

## Особенности

- Добавление, просмотр и удаление напоминаний
- Всплывающие окна поверх всех окон при срабатывании напоминания
- Системные уведомления Windows 11 (работают даже при свёрнутом окне)
- Хранение данных в локальной базе SQLite
- Автоматическая проверка просроченных напоминаний
- Статусы: Ожидает, Готово, Просрочено, Отменено
- Контрастная тёмная тема

## Требования

- Windows 10/11
- Python 3.7+

## Установка

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python main.py
```

## Запуск приложения

### Способ 1: Двойной клик (без окна консоли)
Нажмите дважды на файл **`run_hidden.vbs`**

### Способ 2: Через терминал
```powershell
& ".venv\Scripts\python.exe" main.py
```

### Способ 3: Через bat-файл (с консолью)
Нажмите дважды на файл **`run.bat`**

## Использование

### Добавление напоминания

1. Заполните поля в верхней части окна:
   - **Заголовок** (обязательно)
   - **Описание** (многострочное поле)
   - **Дата и время** (формат: ГГГГ-ММ-ДД ЧЧ:ММ)
2. Нажмите кнопку **"➕ Добавить"**
3. Поля ввода автоматически очищаются

### Редактирование напоминания

1. Выберите напоминание в таблице
2. Нажмите кнопку **"🔄 Обновить"**
3. В открывшемся окне измените данные
4. Нажмите **"Сохранить"**

### Удаление напоминания

1. Выберите напоминание в таблице
2. Нажмите кнопку **"🗑️ Удалить"**
3. Подтвердите действие в диалоговом окне
